Устанавливаю базовые утилиты проверки:
```bash
sudo apt install smartmontools lm-sensors nvme-cli sysstat
```
---
Проверка накопителей:
```bash
lsblk -d -o NAME,TRAN,SIZE,MODEL
```
```bash
sudo smartctl -x /dev/sda | less
```
Для nvme:
```sh
sudo nvme smart-log /dev/nvme0
```
---
Проверка CPU в простое: 
```sh
sensors
```
Стресс-тест цпу(запускать в разных терминалах):
```sh
stress-ng --cpu 0 --timeout 5m --metrics-brief
```
```sh
watch -n 2 sensors
```
---
Память:
```sh
free -h
```
Детальная инфа по модулям:
```sh
sudo dmidecode --type memory
```
Журнал ошибок:
```sh
sudo journalctl -k -b | grep -Ei 'memory error|hardware error|mce|edac|ecc'
```
Тесты:
```sh
sudo memtester 2G 1
```
